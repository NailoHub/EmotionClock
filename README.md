#Emotion Clock
Physical Computing Project 2025 - IT KMITL


วัตถุประสงค์และฟังก์ชันการทำงาน (Objectives and Functionality)

โครงงานนี้มีวัตถุประสงค์เพื่อสร้างระบบนาฬิกาสำหรับคนเหงาที่มีฟังก์ชันการทำงาน 3 ส่วนหลัก ดังนี้:

ระบบนาฬิกา
ใช้ DS3231 Module:ระบบจะใช้ DS3231 Module เพื่อให้นาฬิกาเดินต่อไปเรื่อยๆแม้จะไม่ได้ต่อไฟเลี้ยง(ต้องตั้งก่อนครั้งเเรกหลังจากนั้นไม่ต้อง)

ระบบนาฬิกาปุก
การเตือน:ระบบจะส่งสัญญาณไปให้ buzzer เมื่อการนับถอยหลังเสร็จสิ้น การกดจะทำการเพิ่มเวลาใน alarm clock 1 นาทีต่อการกด 1 ครั้ง

ระบบแสดงคำที่ดีต่อใจ
การ display:มีจอ LCD ติดตั้งอยู่ เพื่อแสดงคำพูดที่ดีๆออกมา คำพูดดีๆ สามารถ manage ได้ที่ เว็ป IP ที่เชื่อมเน็ตกับบอร์ดเเล้วบอร์ดเเสดงออกมา

จากระบบทั้ง 3 จะเเสดงผ่านจอ LCD defult state LCD แถวบนเป็น วันที่ แถวล่างเป็น เวลา, เมื่อระบบนาฬิกาปุกทำงาน จะเปลี่ยนแถวล่างเป็น เวลานับถอยหลัง
และเมื่อกดปุ่มเพื่อใช้ระบบแสดงคำที่ดีต่อใจ แถวบนจะสลับไปมาระหว่าง วันที่/คำที่ดีต่อใจ
